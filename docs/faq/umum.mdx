# Pertanyaan Umum (FAQ)

Temukan jawaban cepat atas pertanyaan yang sering ditanyakan mengenai penggunaan dan kebijakan aplikasi AmalanKu.

## Tentang Aplikasi

### Apa itu AmalanKu?
AmalanKu adalah aplikasi asisten ibadah Muslim yang fokus pada pelacakan amalan harian, statistik progres ibadah, dan privasi data. Aplikasi ini didesain tanpa iklan dan tanpa akun untuk kenyamanan Anda.

### Apakah aplikasi ini gratis?
Ya, AmalanKu dapat diunduh dan digunakan secara gratis. Misi kami adalah membantu umat Muslim untuk lebih istiqomah dalam menjalankan amalan harian.

### Apakah saya perlu koneksi internet?
Hampir seluruh fitur AmalanKu dapat berjalan tanpa internet (Offline). Internet hanya diperlukan saat Anda pertama kali mengunduh aplikasi, melakukan update, atau ingin membackup data ke cloud secara manual.

---

## Privasi & Akun

### Mengapa tidak ada fitur Login?
Kami mengutamakan privasi. Dengan tidak adanya sistem login, data Anda tidak tersimpan di server kami, sehingga identitas dan catatan ibadah Anda sepenuhnya aman di tangan Anda sendiri.

### Apakah pengembang bisa melihat data saya?
Sama sekali tidak. Seluruh database disimpan secara lokal di memori internal ponsel Anda dalam folder yang terenkripsi oleh sistem Android.

---

## Fitur & Teknis

### Bagaimana cara memindahkan data ke HP baru?
Karena tidak ada akun, Anda harus melakukan backup manual:
1. Buka **Fitur Baru > Backup & Restore** di HP lama.
2. Simpan file cadangan (ZIP).
3. Pindahkan file tersebut ke HP baru.
4. Pilih **Restore** di aplikasi AmalanKu pada HP baru.

### Mengapa waktu adzan di aplikasi berbeda dengan di Masjid?
Perbedaan 1-2 menit sering terjadi karena perbedaan metode perhitungan atau titik koordinat GPS. Anda dapat menyesuaikannya melalui menu **Pengaturan > Notifikasi > Koreksi Waktu**.

---

## Troubleshooting Singkat

> [!IMPORTANT]
> **Data saya hilang setelah ganti HP!**
> Karena AmalanKu tidak menggunakan server cloud otomatis, data Anda tidak akan pindah kecuali Anda melakukan proses [Backup & Restore](/fitur-baru/backup-restore) secara manual.

> [!TIP]
> **Aplikasi terasa lambat?**
> Pastikan Anda memiliki ruang penyimpanan yang cukup di memori internal dan cobalah untuk membersihkan *cache* aplikasi melalui pengaturan sistem Android.
