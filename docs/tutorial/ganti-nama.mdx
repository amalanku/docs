# Cara Mengubah Nama

Personalisasikan akun Anda dengan mengubah nama tampilan agar pengalaman penggunaan aplikasi terasa lebih personal.

### Langkah-langkah Mengubah Nama:

Anda dapat mengubah nama melalui dua cara mudah berikut ini:

1.  **Melalui Ringkasan Bulanan:**
    * Buka menu utama aplikasi.
    * Cari bagian **Ringkasan Bulanan**.
    * Ketuk pada teks yang bertuliskan: **"Ubah nama kamu di sini"**.

2.  **Melalui Menu Pengaturan:**
    * Masuk ke menu **Pengaturan** (Settings).
    * Pilih opsi profil atau pengaturan akun.
    * Ketuk pada kolom nama yang tersedia.

---
<Note>
**Pembaruan Otomatis:** Begitu Anda selesai mengubah teks nama, data akan langsung tersimpan secara otomatis tanpa perlu menekan tombol simpan tambahan.
</Note>
