# Menambahkan Widget Shalat

Pantau waktu shalat dengan lebih mudah tanpa harus membuka aplikasi melalui fitur Widget di layar utama (*Home Screen*) Anda.

### Langkah-langkah Memasang Widget:

1.  **Masuk ke Mode Edit Layar:**
    * Pergi ke **Home Screen** (layar utama) HP Anda.
    * Tekan dan tahan pada area kosong di layar hingga muncul menu pengaturan tampilan.
2.  **Buka Menu Widget:**
    * Ketuk ikon atau tulisan **Widget**.
3.  **Cari Nama Aplikasi:**
    * Gulir ke bawah hingga Anda menemukan nama aplikasi AmalanKu, atau gunakan kolom pencarian dengan mengetik nama aplikasi.
4.  **Pilih Widget Shalat:**
    * Temukan pilihan **Widget Waktu Shalat**.
    * Tekan dan tahan widget tersebut, lalu geser (drag) ke posisi layar yang Anda inginkan.
5.  **Selesai:**
    * Lepaskan jari Anda untuk meletakkan widget. Kini jadwal shalat akan selalu terlihat setiap kali Anda membuka HP.

---

<Note>
**Akses Cepat:** Anda bisa langsung mengetuk widget tersebut di layar utama untuk masuk ke halaman detail jadwal shalat atau mengatur notifikasi adzan.
</Note>
