# Backup & Restore Data

Gunakan fitur ini untuk memastikan data amalan, catatan, dan pengaturan Anda tetap aman meskipun Anda berpindah perangkat atau melakukan instalasi ulang.

### Cara Mencadangkan (Backup) Data:

1.  **Buka Pengaturan:** Masuk ke menu **Pengaturan** dan pilih menu **Backup & Restore**.
2.  **Pilih Backup:** Ketuk tombol **Backup Data**.
3.  **Input Sandi:** Anda akan diminta memasukkan sandi untuk mengunci file cadangan. *Ingat sandi ini baik-baik!*
4.  **Konfirmasi Sukses:** Jika berhasil, modal sukses akan muncul.
5.  **Lokasi File:** File cadangan Anda disimpan di memori internal pada folder:
    `documents/amalanku/`
    dengan format nama: `amalanku_backup_yyyy-mm-dd.zip`.

---

### Cara Memulihkan (Restore) Data:

1.  **Pilih Menu Restore:** Di halaman yang sama, pilih opsi **Restore Data**.
2.  **Verifikasi Sandi:** Masukkan sandi yang Anda buat saat proses backup tadi.
    * **Jika sandi salah:** Aplikasi akan memunculkan pesan error dan proses dibatalkan.
    * **Jika sandi benar:** Modal sukses akan muncul.
3.  **Auto Restart:** Setelah menekan OK pada modal sukses, **aplikasi akan otomatis restart** untuk memuat ulang seluruh data yang telah dipulihkan.

---

> **Keamanan File:** Jangan menghapus folder `amalanku` di dokumen ponsel Anda jika ingin melakukan restore di kemudian hari. Pastikan sandi yang Anda masukkan saat restore sama persis (case-sensitive) dengan saat melakukan backup.
