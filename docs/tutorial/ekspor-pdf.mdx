# Ekspor Laporan PDF

Simpan progres ibadah Anda dalam bentuk dokumen digital atau cetak laporan untuk evaluasi pribadi menggunakan fitur Ekspor PDF.

### Cara Membuat Laporan PDF:

1.  **Buka Pengaturan:**
    * Masuk ke aplikasi dan pilih menu **Pengaturan** (Settings).
2.  **Pilih Menu Ekspor PDF:**
    * Cari dan ketuk opsi bertuliskan **Ekspor Laporan PDF**.
3.  **Buat Laporan Baru:**
    * Perhatikan di layar tersebut, temukan **Tombol FAB** (tombol bulat bertanda **plus `+`**) yang berada di **pojok kanan bawah** layar.
    * Ketuk tombol tersebut untuk mulai membuat dokumen baru.
4.  **Proses Selesai:**
    * Sistem akan memproses data Anda secara otomatis (*PDF Generated*).
    * Setelah muncul pratinjau, laporan Anda siap untuk **disimpan** ke perangkat atau langsung **dibagikan (share)** ke media sosial atau aplikasi pesan.

---

> **Laporan Berkala:** Anda disarankan mengekspor laporan setiap akhir bulan sebagai arsip pribadi untuk melihat perkembangan konsistensi ibadah Anda dari waktu ke waktu.
