# Kustomisasi Profil

Meskipun AmalanKu bersifat **Tanpa Akun**, Anda tetap dapat mempersonalisasi aplikasi agar terasa lebih dekat dengan Anda. Fitur kustomisasi profil ini berfungsi untuk memberikan identitas visual pada laporan dan dashboard Anda.

## Mengapa Melakukan Kustomisasi?

* **Personalisasi:** Memberikan sentuhan personal pada aplikasi ibadah harian Anda.
* **Laporan PDF:** Nama yang Anda masukkan akan muncul di bagian header saat Anda melakukan [Ekspor Laporan PDF](/tutorial/ekspor-pdf).
* **Motivasi:** Melihat nama atau target pribadi di dashboard dapat meningkatkan semangat dalam beramal.

---

## Hal-Hal yang Dapat Diatur

Anda dapat menyesuaikan beberapa elemen profil berikut:

1.  **Nama Panggilan:** Nama yang akan muncul di layar utama (misal: "Selamat pagi, Abdullah").
2.  **Foto Profil:** Mengambil foto dari galeri atau kamera sebagai avatar.
3.  **Moto Hidup/Target:** Kalimat motivasi singkat yang akan muncul di dashboard sebagai pengingat tujuan ibadah Anda.

---

## Cara Mengubah Profil

Ikuti langkah mudah berikut:

1.  Buka menu **Pengaturan**.
2.  Pilih sub-menu **Kustomisasi Profil**.
3.  Klik pada ikon kamera untuk mengubah foto atau klik pada kolom teks untuk mengubah nama.
4.  Tekan tombol **Simpan** untuk menerapkan perubahan.

---

## Keamanan Data Profil

Sama seperti data amalan, data profil Anda diperlakukan dengan standar privasi yang sama:

> **Hanya Lokal:** Foto dan nama yang Anda unggah **tidak akan dikirim ke server mana pun**. File tersebut tetap berada di folder internal aplikasi di ponsel Anda untuk menjaga privasi penuh.

> **Nama Samaran:** Karena ini hanya untuk kenyamanan Anda sendiri, Anda bebas menggunakan nama asli maupun nama samaran (Kunya) sesuai keinginan.
---

## Pertanyaan Sering Diajukan (FAQ)

**Apakah foto profil saya akan hilang jika saya ganti HP?**
Ya, karena data bersifat lokal. Jika Anda pindah perangkat, Anda perlu mengatur ulang foto profil di perangkat baru, atau memastikan folder media ikut tercadangkan saat melakukan backup manual.

**Apakah nama profil mempengaruhi fitur notifikasi?**
Ya, jika diaktifkan, notifikasi pengingat mungkin menyapa Anda menggunakan nama yang telah Anda atur agar terasa lebih personal.
