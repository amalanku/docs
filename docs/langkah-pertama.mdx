## âœ¨ Tips Memulai

### Minggu Pertama (Orientation)

**Hari 1-3: Kenali Interface**
- Jelajahi semua tab
- Coba setiap fitur dasar
- Biasakan diri dengan navigasi
- Jangan takut untuk mencoba-coba

**Hari 4-7: Mulai Tracking Rutin**
- Mark shalat setiap hari (wajib dulu)
- Cek daily task minimal 1x sehari
- Catat minimal 1 kebaikan/hari
- Lihat statistik harian

### Minggu Kedua (Build Routine)

**Tingkatkan Konsistensi:**
- Aktifkan notifikasi reminder
- Pasang widget di home screen
- Buat backup pertama
- Mulai catat keburukan untuk muhasabah
- Review progress mingguan

### Minggu Ketiga (Habit Formation)

**Bangun Kebiasaan:**
- Jadikan tracking sebagai rutinitas
- Review statistik mingguan secara rutin
- Pertimbangkan naik level jika sudah konsisten
- Coba fitur lanjutan seperti kekhusyukan shalat

## â“ Pertanyaan Umum Pemula

**Q: Harus menyelesaikan semua daily task?**
A: Tidak harus. Fokus pada konsistensi amalan dasar dulu. Lebih baik menyelesaikan 80% secara rutin daripada 100% tapi hanya sesekali.

**Q: Bagaimana jika lupa mark shalat?**
A: Bisa mark di hari yang sama hingga tengah malam. Setelah itu akan masuk hari baru. Jangan khawatir jika terlewat, fokus ke hari berikutnya.

**Q: Data akan hilang jika uninstall?**
A: Ya, karena data lokal. Makanya penting buat backup rutin! Backup minimal 1x/bulan atau sebelum uninstall/update.

**Q: Bisa digunakan offline?**
A: Ya, 100% offline. Internet hanya untuk download app. Semua fitur berjalan tanpa koneksi.

**Q: Bagaimana cara ubah level task?**
A: Settings â†’ Level Daily Task â†’ Pilih level baru (Esensial/Berkembang) â†’ Save. Bisa diubah kapan saja.
