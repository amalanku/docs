# Tanpa Akun & Registrasi

Salah satu hambatan terbesar dalam menggunakan aplikasi baru adalah proses pendaftaran yang rumit. Di **AmalanKu**, kami menghilangkan hambatan tersebut. Anda bisa langsung menggunakan aplikasi sesaat setelah instalasi selesai tanpa perlu mendaftar sama sekali.

## Filosofi "Langsung Pakai"

Kami percaya bahwa untuk memulai kebaikan, tidak boleh ada penghalang. 
* **Bebas Email/No. HP:** Kami tidak meminta alamat email, nomor telepon, atau sinkronisasi akun media sosial.
* **Bukan Produk:** Kami tidak mengumpulkan profil pengguna untuk kepentingan pemasaran atau iklan.
* **Fokus pada Ibadah:** Anda tidak perlu mengingat satu lagi *username* atau *password* akun online.

---

## Keuntungan Bagi Pengguna

### 1. Anonimitas Total
Karena tidak ada akun, identitas asli Anda tidak pernah terhubung dengan data ibadah Anda. Siapa Anda, di mana Anda berada, dan bagaimana kualitas ibadah Anda tetap menjadi rahasia antara Anda dan Allah SWT.

### 2. Keamanan dari Kebocoran Data
Dalam beberapa tahun terakhir, banyak kasus kebocoran data server (data breach) pada layanan besar. Karena AmalanKu tidak memiliki server pusat berisi data pengguna, maka data Anda **mustahil** bocor dari pihak kami.

### 3. Setup Instan
Cukup unduh, buka, dan mulai catat amalan pertama Anda. Cocok bagi pengguna yang mengutamakan efisiensi dan kesederhanaan.

---

## Bagaimana Jika Saya Berganti HP?

Karena data tidak terikat pada akun cloud (seperti login Google), maka perpindahan data dilakukan secara mandiri:

1.  **Ekspor Manual:** Gunakan fitur [Backup & Restore](/tutorial/backup-restore) di HP lama.
2.  **Pindahkan File:** Kirim file backup tersebut ke HP baru Anda.
3.  **Impor:** Gunakan fitur Restore di HP baru untuk memulihkan semua riwayat amalan Anda.

---

## Tips Penggunaan

> **Identitas Perangkat:** Meskipun tidak ada akun, aplikasi mengenali database berdasarkan instalasi di perangkat tersebut. Menghapus data aplikasi (Clear Data) melalui pengaturan sistem Android akan menghapus seluruh catatan Anda secara permanen jika belum di-backup.

> **Kustomisasi Profil:** Anda tetap bisa mengatur nama panggilan atau foto profil di dalam aplikasi (hanya untuk tampilan visual di perangkat Anda) melalui menu [Kustomisasi Profil](/pengaturan/profil).

## Kesimpulan
AmalanKu adalah alat (tools) milik Anda sepenuhnya. Kami hanya menyediakan wadahnya, andalah yang memegang kuncinya. Tanpa akun berarti tanpa jejak digital yang tidak perlu.
